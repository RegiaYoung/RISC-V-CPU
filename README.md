# RISC-V CPU

记录一下夏季学期计算机设计与实践课上写的RISC-V单周期CPU和RISC-V五级流水线CPU



## 已实现功能

* miniRV-1指令集
* 流水线CPU的停顿、前递、静态分支预测



## 说明

* 开发板

  Minisys开发板 (芯片型号：**XC7A100TFGG484-1**)

* 实现频率

  单周期CPU：25MHz

  流水线CPU：70MHz
